# Context panel stability v0.28.5

- The context explanation disclosure now retains its open state across live HUD rerenders.
- Disclosure state is scoped to the selected object or build tool and resets when the context target changes.
- Context metric cells use a fixed three-column grid on portrait, landscape and target panels.
- Metric padding is reduced to lower the panel height without removing information.
- Save schema and game simulation rules are unchanged.
