# FRONTLINE ROADS v0.32.10 ブラウザ・実機確認項目

## 自動環境で確認済み

- HTML、CSS、JavaScript、Service WorkerのローカルHTTP応答。
- 表示品質の保存・復元ロジック。
- モーダルCSSにblur、品質依存filter、品質依存clip-pathがないこと。
- 背景タップとEscapeによる閉鎖処理。

## 公開HTTPS環境で確認する項目

1. Android端末でMENUを開く。
2. 表示品質を省電力、標準、高精細の順に切り替える。
3. 各品質でメニューカードと×ボタンが消えないことを確認する。
4. MENUを閉じ、文明・拠点・敵への派兵画面を開く。
5. 各画面が暗い背景だけにならず、内容と操作ボタンを表示することを確認する。
6. 暗い背景部分をタップして閉じられることを確認する。
7. 再読み込み後も保存された表示品質で操作可能なことを確認する。
8. Service Worker更新後、画面下部が`0.32.10-modal-display-recovery`へ切り替わることを確認する。

## 環境制限

作業環境のChromiumはローカルURLへの遷移を管理ポリシーで遮断したため、実ブラウザDOMとGPU合成の自動スクリーンショットは取得できませんでした。構文、CSS不変条件、DOMイベント、HTTP資産、全回帰は独立して検証しています。
