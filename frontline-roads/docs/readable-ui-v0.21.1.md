# FRONTLINE ROADS v0.21.1 — Readable UI

This maintenance release increases the smallest HUD and modal text while preserving the radar layout and gameplay scale.

## Changes

- Explicit CSS type sizes below 15 px increased by approximately 1–2 px.
- Minimum explicit CSS type size is now 8 px instead of 6 px.
- Canvas labels increased by 1 px independently of map zoom.
- Construction tool controls enlarged from 72 px to 78 px tall.
- Context panel clearance increased so the larger toolbar does not cover information.
- Threat-status width increased to reduce wrapping.

No save schema, combat values, economy values, road data or location rules changed.
